# AI Mechanic Mobile

Expo-based React Native client for the existing NestJS backend in `../Back`.

## Run

1. `cd Mobile`
2. `npm install`
3. Set `EXPO_PUBLIC_API_BASE_URL` if your phone cannot reach `localhost:3000`
4. `npm start`

## Default API URLs

- Android emulator: `http://10.0.2.2:3000`
- iOS simulator / web: `http://localhost:3000`

## Included Screens

- Home dashboard
- AI diagnosis upload
- Diagnosis history with linked quotes and OBD logs
- Pricing search
- Quote fairness checker
- OBD analyzer
